About this project.

Terminal Velocity is a web application that is reminiscent of text-adventure games of the 80's.
Inspired by: Alien, Zork, Stories Untold and 80's aesthetics with a modern twist.

Built on a modern web platform it combines the publication strength of the web, using it to reach a bigger audience.

Jekyll is a static site generator written in Ruby, that rendors Markdown and Liquid templates. Jekyll powers GitHub Pages - public web pages are freely hosted on GitHub's domain.
